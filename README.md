# NeoVenturi — 3D Viewer

Interactive web viewer for a compressed-air Venturi vacuum generator, built with Next.js and React Three Fiber.

## Features

- **Info tab** — high-level design brief: operating principle, baseline geometry, and CFD design variables
- **Model tab** — interactive 3D viewer with:
  - Orbit, pan, and zoom controls
  - Wireframe toggle
  - Transparency toggle
  - Section cut with adjustable height slider
  - Fullscreen mode
  - Axis gizmo and infinite grid

## Stack

- [Next.js 14](https://nextjs.org/) (App Router)
- [React Three Fiber](https://docs.pmnd.rs/react-three-fiber)
- [Three.js](https://threejs.org/)
- [@react-three/drei](https://github.com/pmndrs/drei)

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
